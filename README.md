## Run application:

My current portfolio is available at: [https://enzobozzani.vercel.app](https://enzobozzani.vercel.app)

Clone project:

```
git clone https://github.com/EnzoBozzani/portfolio2.0.git
```

Install dependencies:

```
npm install
```

Run project:

```
npm run dev
```
